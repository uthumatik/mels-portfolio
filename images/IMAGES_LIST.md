## Conseils pour les images

1. **Format** : JPEG pour les photos, PNG pour les logos avec transparence
2. **Optimisation** : Compressez vos images (TinyPNG, Squoosh, etc.)
3. **Qualité** : Utilisez des images haute résolution
4. **Cohérence** : Gardez un style visuel cohérent

## Télécharger les images depuis Canva

1. Allez sur votre portfolio Canva
2. Cliquez sur chaque image
3. Téléchargez en haute qualité
4. Renommez selon la liste ci-dessus
5. Placez dans le dossier `images/`
